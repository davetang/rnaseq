## README

Use [mamba](https://github.com/conda-forge/miniforge#mambaforge) to install and
create environments.

```console
mamba env create -f jellyfish.yml 
mamba env create -f rnaseq.yml 
```

Activate to use.

```console
mamba activate rnaseq
```

Deactivate when finished.

```console
mamba deactivate
```
